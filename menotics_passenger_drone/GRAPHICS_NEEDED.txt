# Graphics Assets Needed for Menotics Passenger Drone Mod

## Required Texture Files (6 PNG files total)

Place these files in the `textures/` folder of the mod:

### 1. menotics_drone_front.png
- **Purpose**: Front face of the drone (narrow end, shows cockpit/glass area)
- **Recommended Size**: 64x64 or 128x128 pixels
- **Design Notes**: 
  - Should show a glass/metal frame design
  - Include transparent or semi-transparent areas to show passengers inside
  - Consider adding lights or indicators
  - This is the narrow front face (drone is 4 blocks long, this is the 2-block wide end)

### 2. menotics_drone_back.png
- **Purpose**: Rear face of the drone (narrow end)
- **Recommended Size**: 64x64 or 128x128 pixels
- **Design Notes**:
  - Could show engines, thrusters, or exhaust
  - Match the style of the front texture
  - Same dimensions as front (narrow 2-block wide face)

### 3. menotics_drone_side.png
- **Purpose**: Both left and right sides of the drone (same texture used for both)
- **Recommended Size**: 128x64 or 256x128 pixels (wider since drone is 4 blocks long)
- **Design Notes**:
  - This is the main visible surface when drone is moving
  - Show large glass panels to see passengers inside
  - Add structural frames between glass sections
  - Consider adding windows divided into sections matching the 4-block length
  - Include details like doors, vents, or decorative elements

### 4. menotics_drone_roof.png
- **Purpose**: Top surface of the drone
- **Recommended Size**: 128x64 or 256x128 pixels (4 blocks long, 2 blocks wide)
- **Design Notes**:
  - Show the roof structure
  - Could include antennas, lights, or ventilation
  - Match the metallic/glass aesthetic
  - Same proportions as the side (4 blocks long x 2 blocks wide)

### 5. menotics_drone_bottom.png
- **Purpose**: Bottom/underside of the drone
- **Recommended Size**: 128x64 or 256x128 pixels (4 blocks long, 2 blocks wide)
- **Design Notes**:
  - Show landing gear, thrusters, or mechanical details
  - Could have a darker, more industrial look
  - Same proportions as the side (4 blocks long x 2 blocks wide)

## Important Notes

### No 3D Model Required!
This mod uses Minetest's built-in "cube" visual type with custom dimensions.
The drone will be rendered as a stretched box with your textures applied to each face.
Dimensions: 4 blocks long (X), 3 blocks high (Y), 2 blocks wide (Z)

### Glass Effect for Passengers
To make passengers visible:
- Use semi-transparent PNG files with alpha channel
- The glass areas should be partially transparent (alpha ~100-150)
- Consider using a tinted blue or cyan color for the glass
- Frame the glass with opaque metal borders
- Players inside will be visible through the transparent parts!

### Terminal Block
- **terminal.png** already exists in the default game textures (as you mentioned)
- No additional graphics needed for terminals

## Suggested Color Scheme
- Primary: Metallic gray/silver for frames
- Secondary: Light blue or cyan tint for glass panels (semi-transparent!)
- Accent: Orange or red for lights/indicators
- Dark gray for underside/mechanical parts

## Texture Mapping on Cube
The cube visual maps textures in this order:
1. Right side (menotics_drone_side.png)
2. Left side (menotics_drone_side.png) - same texture
3. Top/Roof (menotics_drone_roof.png)
4. Bottom (menotics_drone_bottom.png)
5. Front (menotics_drone_front.png)
6. Back (menotics_drone_back.png)
