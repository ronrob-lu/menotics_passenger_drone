# Graphics Assets Needed for Menotics Passenger Drone Mod
==========================================================

All textures should be 84x64 pixels as specified.
Place these PNG files in the textures/ folder.

## Drone Textures (5 files required):

1. menotics_drone_front.png (84x64)
   - Front view of the drone (+X face)
   - Should show glass panels so passengers are visible inside
   - Consider adding headlights or front details

2. menotics_drone_back.png (84x64)
   - Back view of the drone (-X face)
   - Should show glass panels so passengers are visible inside
   - Consider adding rear lights, exhaust, or propeller details

3. menotics_drone_side.png (84x64)
   - Side view of the drone (used for both +Z right and -Z left faces)
   - Should show glass panels so passengers are visible inside
   - This texture appears on both sides of the drone
   - Consider adding windows, doors, or side details

4. menotics_drone_roof.png (84x64)
   - Top view of the drone (+Y face)
   - Can be solid or have glass panels
   - Consider adding roof details, antennas, or vents

5. menotics_drone_bottom.png (84x64)
   - Bottom view of the drone (-Y face)
   - Usually solid with mechanical details
   - Consider adding landing gear shadows, thrusters, or mechanical details

## Important Notes:

- Use semi-transparent areas (alpha channel) in glass panel areas so players can see passengers inside
- The drone is 4 blocks long, 3 blocks high, and 2 blocks wide
- Textures are mapped to a cube visual with visual_size {x=4, y=3, z=2}
- Glass areas should allow viewing of passengers seated inside

## Terminal Block:

- terminal.png - Already exists in the default textures folder, no action needed!

## Texture Mapping Order (for cube visual):
The cube visual maps textures in this exact order:
1. Front (+X)  -> menotics_drone_front.png
2. Back (-X)   -> menotics_drone_back.png
3. Right (+Z)  -> menotics_drone_side.png
4. Left (-Z)   -> menotics_drone_side.png (same file)
5. Top (+Y)    -> menotics_drone_roof.png
6. Bottom (-Y) -> menotics_drone_bottom.png
